# US Chess Tournament Directory

A lightweight web app for discovering chess tournaments across the United States.

## Features

- Daily-refreshing tournament data cache (24 hours).
- Source catalog listing (federation + organizers + aggregators).
- Tournament cards with key details.
- Click through to a tournament details page.
- Filters:
  - by US state
  - by tournaments within 100 miles of a city

## Run locally

```bash
python3 -m http.server 8000
